Set Dot matrix display

Here you can edit dot matrix display coefficients.
To select the next editable coefficient, press the button labelled with the current coefficient name.
Play with the slider to change its value or key in a value.
You can provide preset coefficients.
Exit this screen to return to the Single Chrono/Timer screen.
