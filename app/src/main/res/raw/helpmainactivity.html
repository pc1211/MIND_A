Mastermind

Welcome to Matermind !
This is what you see on the main screen:
  • The selection of who guesses: User (you) or Android (your phone)
  • The available color palette
  • The current proposal and its score
  • The possible actions
  • The proposal and score history list
If User guesses
  • Fill the current proposal by selecting colors from the palette, history list or current proposal
    (you can cheat by pressing)
  • You can clear one or more colors of the current proposal by pressing
  • Press on the current proposal dot matrix display to get its score and move the current proposal to the history list
If Android guesses
  • Press on the current proposal dot matrix display to enter the score for the current proposal made by Android
  • Once in the Score Activity, enter the number of blacks and whites by pressing the radio buttons
  • Then the current proposal is moved to the history list and Android makes another guess
At any moment, you can:
  • Delete the last created item in the history list by pressing
  • Start a new game by pressing
A Menu is also available:
  • Number ofPegs: Enter the new number of pegs (1 to 9); this will start a new game
  • Number ofColors: Enter the new number of colors (1 to 10); this will start a new game
  • Edit Palette: Change the default colors and store presets
  • Help: Display help
  • About: Display program version
Press the light bulb icon on the menu bar to keep screen on/off
